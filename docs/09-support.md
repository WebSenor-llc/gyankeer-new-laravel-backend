## How to raise a support ticket

### Self-service first (often fastest)

1. Check this documentation
2. Search the FAQ ([08-faq.md](08-faq.md))
3. Check the troubleshooting guide if present

### Filing a ticket

Email **support@websenor.com** with:

- **Subject:** brief problem statement
- **Account:** company name + Hreasy URL
- **Severity:** P1 / P2 / P3 / P4 (your assessment)
- **What happened:** step-by-step
- **What you expected:** the correct behaviour
- **Screenshot / video:** if visual
- **Browser:** Chrome 120, Safari 17, etc.
- **Logs / error message:** if any

We respond per the SLA above.

### Emergency P1 (production down)

For Professional and Enterprise customers:
- Email support@websenor.com with `[P1]` in subject
- Call your CSM phone (provided at signup)
- WhatsApp: +91 XXXX-XXX-XXX

Cloud customers: we proactively monitor and may already be on it.
