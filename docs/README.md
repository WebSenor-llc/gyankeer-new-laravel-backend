# Hreasy by WebSenor

> Cloud-ready HR & Payroll for India — built for factories, manufacturing, and contract-labour-heavy businesses. Compliant with the new Code on Social Security 2020.

## What is Hreasy?

Hreasy is a complete HR and payroll platform that automates everything from
hiring to monthly salary disbursement, while keeping you 100% compliant with
Indian statutory requirements (EPF, ESI, PT, LWF, TDS, Bonus, Gratuity,
POSH, and the new Code on Social Security 2020).

It replaces the spreadsheets, the ageing on-premise software, and the three
different vendors most businesses currently juggle — with one unified
multi-company portal that scales from 50 to 5,000+ employees.

## Who is it for?

- **Factory & manufacturing units** with mixed staff + contract labour
- **Multi-entity groups** running 2–10 sister concerns under one HR team
- **Mid-sized businesses** (50 to 5,000 employees) outgrowing Excel + Tally Payroll
- **HR consultants** managing payroll for multiple clients
